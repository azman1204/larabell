GIT
=====
www.github.com
www.git-scm.com
> git clone https://github.com/azman1204/larabell.git
> git status
 - list semua files yg baru atau yg ada update
> git add *
 - add semua file ke dlm list utk di trace versioning
> git reset head <nama file>
 - undo file yg telah di add
> git commit -m "....."
 - start trace versioning
> git push
